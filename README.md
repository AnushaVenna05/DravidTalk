# DravidaTalk - Learn Malayalam, Feel Kerala 🌴💬

A premium, culturally-focused Chrome extension designed to help you master Malayalam through Telugu or English. Experience the essence of Kerala while you learn, featuring AI-powered translations, phonetic guides, and fascinating cultural insights.

## ✨ Features
- **Instant AI Translation**: Powered by Gemini 2.5 Flash for high accuracy and cultural nuances.
- **Phonetic Transliteration**: Clear guides for Telugu and English speakers to pronounce Malayalam perfectly.
- **Did You Know?**: Daily fascinating facts about Kerala's history, food, and geography.
- **Smart Audio Proxy**: Native-sounding pronunciations that work on **any website** by bypassing CSP restrictions.
- **Cultural UI**: A beautiful "Nilavilakku" inspired theme (Forest Green & Antique Gold).
- **Auto-Reset Workflow**: Clears your workspace when you return home for a seamless learning experience.

## 🚀 Getting Started

### 1. API Key Setup (Stay Free!)
To keep this tool free for everyone, DravidaTalk uses your own personal Gemini API key.
1. Visit [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Click **Create API Key**.
3. Create a `.env` file in the root directory.
4. Add your key: `VITE_GEMINI_API_KEY=your_actual_key_here`.

### 2. Build the Extension
Ensure you have Node.js installed, then run:
```bash
npm install
npm run build
```

### 3. Install in Chrome
1. Go to `chrome://extensions/` in your browser.
2. Turn on **Developer mode** (toggle in the top right).
3. Click **Load unpacked**.
4. Select the `dist` folder generated by the build.
5. Pin **DravidaTalk** to your toolbar for the best experience!

## 🛠 Tech Stack
- **Framework**: React 19 + TypeScript
- **Build System**: Vite + @crxjs/vite-plugin
- **AI Engine**: Google Gemini 2.5 Flash
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Custom Design System)
- **Communications**: Chrome Extension Messaging (IPC) for Background/Content script synchronization.
